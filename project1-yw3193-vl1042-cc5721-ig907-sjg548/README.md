# How To Run:  
### Requires:
* Python (Flask, json, sqlite3)
* sqlite3

### To Start:
In your command console, type:  
```  
python app.py  
```  

If you can't connect to the host when running application, change:  
```python  
app.run(host='0.0.0.0')
```  
to:  
```python  
app.run()
```  

### To Reset Database:    
```  
cd /data  
sudo rm database.db  
sqlite3 database.db  
CREATE TABLE ToDoList ('index' INTEGER PRIMARY KEY, 'todo' TEXT);
```
